# chargeback-process
Описание и моделирование процесса «Урегулирование спорной транзакции (chargeback)»
Бизнес-контекст: Chargeback — сложный, регламентированный процесс. Участвуют 4 стороны. Жёсткие сроки: 120 дней на оспаривание, 45 дней на ответ. Аналитик должен описать процесс с учётом всех таймингов.

Артефакты работы:

1) Use Cases: «Клиент оспаривает транзакцию», «Оператор рассматривает диспут», «Система автоматически возвращает деньги» (для сумм < 3000 ₽).
2) BPMN-схема полного процесса (пулы: Клиент, Банк-эквайер, Платёжная система, Банк-эмитент).
3) Описание интеграции с платёжной системой (Mastercard/Visa): протокол, формат, гарантии доставки.
4) REST API POST /api/v1/disputes (создание диспута клиентом). Метод, URL, тело запроса/ответа, коды ответов. Спецификация API.
